# burgerhub-website
BurgerHub is a simple fast-food restaurant website built with HTML, TailwindCSS, and JavaScript.
Users can browse menus, add items to the cart, and simulate ordering burgers online.

This project demonstrates a basic front-end ordering system for a burger restaurant.

---

# Features
- Browse burger menu and combo sets  
- Add items to cart  
- Increase / decrease quantity  
- Remove items from cart  
- Select payment method  
- Generate receipt after order 

---

# Technologies Used
- HTML5  
- TailwindCSS  
- JavaScript
